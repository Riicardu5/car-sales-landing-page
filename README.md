# 🚗 Car Sales Landing Page - Template

Este é um projeto de uma **Landing Page de alta conversão** para venda de veículos particulares. O objetivo foi criar uma interface simples, rápida e totalmente responsiva, focada em facilitar o contato direto via WhatsApp.

## 🛠️ Tecnologias Utilizadas

* **HTML5**: Estrutura semântica.
* **Tailwind CSS**: Estilização moderna e utilitária (via CDN).
* **FontAwesome**: Ícones para melhor experiência visual.
* **Mobile First**: Design otimizado para smartphones, onde ocorre a maioria dos acessos.

## ✨ Funcionalidades

- [x] Cabeçalho de destaque "Vende-se".
- [x] Seção de especificações técnicas organizada.
- [x] Badge de destaques para itens opcionais e acessórios.
- [x] Botão flutuante/destacado de WhatsApp para conversão rápida.
- [x] Galeria de fotos responsiva.
- [x] Design limpo e profissional.

## 🚀 Como usar este template

1. Faça o clone do repositório ou baixe o arquivo `index.html`.
2. Substitua os links de imagens `src` pelas fotos reais do seu veículo.
3. Edite os textos com as informações (Modelo, KM, Ano, Opcionais).
4. No botão do WhatsApp, altere o link `https://wa.me/55...` para o seu número.
5. Faça o deploy em plataformas como **Netlify**, **Vercel** ou **GitHub Pages**.
